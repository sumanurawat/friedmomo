Privacy

This is the whole thing: Storyboarder runs on your computer. Your stories don't leave it.

What stays local

  • Every storyboard you make
  • Every character and location you create
  • Every image the app generates
  • Your OpenRouter API key
  • Your chat history with the assistant

All of this lives in a folder called Storyboarder inside your home directory. You can back it up, open the files in a text editor, or copy them to another computer. They're yours.

What touches the network

When you ask Storyboarder to write something or generate an image, the app sends your prompt from your computer directly to OpenRouter's servers (the AI platform you set up during first-run). It does this using your API key. We don't run any server in between.

That means OpenRouter sees your prompts — the same way they would if you used the OpenRouter website directly. Their privacy policy is at openrouter.ai/privacy.

What we don't collect

We don't have analytics. We don't have crash reporting. We don't have a login. We don't have a database. The Storyboarder app doesn't phone home to friedmomo.com for any reason, ever.

The only exception is the update check — once per app launch, the app asks GitHub "is there a new version?" and GitHub responds yes or no. This is standard for apps that auto-update, and GitHub sees your IP address during that request.

If you email us

If you send us an email, we'll see your email address and whatever you said. We keep that message only long enough to reply, unless you explicitly ask to be added to an update list.
